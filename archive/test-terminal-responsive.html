Terminal Responsive Test

Test Overview

This test verifies that the terminal component maintains a fixed height and prevents layout shifts during typing animations.

Resize your browser window to test different screen sizes:

  • Desktop: ≥ 769px (300px terminal height)
  • Tablet: ≤ 768px (250px terminal height)
  • Mobile: ≤ 480px (200px terminal height)
